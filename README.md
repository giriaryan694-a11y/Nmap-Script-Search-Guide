# Nmap NSE Script Notes

This repository contains educational notes for Nmap's NSE (Nmap Scripting Engine) scripts. It is designed for learning and practical reference for penetration testers and students.

Live site: [https://giriaryan694-a11y.github.io/Nmap-Script-Search-Guide/](https://giriaryan694-a11y.github.io/Nmap-Script-Search-Guide/)

## Features

- Clean, educational notes on Nmap NSE
- Types of scripts explained (Discovery, Vulnerability, Brute)
- Searching scripts using `cat` & `grep`
- NSE script categories
- Writing your first NSE script example
- Interactive sidebar navigation
- Search filter for code blocks
- Copy buttons for commands
- Dark/Light theme toggle with preference saved in cookies
- Fully responsive for desktop and mobile screens

## GitHub Pages

This repository is ready to be published as a GitHub Pages site. Steps:

1. Clone or fork the repository.
2. Ensure `index.html` is in the root of the repository.
3. Go to **Settings > Pages**.
4. Under **Source**, select `main` branch and root folder.
5. Click **Save**.
6. Your site will be available at `https://<your-username>.github.io/<repo-name>/`

## Usage

- Open `index.html` in a browser to view notes.
- Click the **hamburger menu** (top-left) to open navigation.
- Click **Toggle Theme** (top-right) to switch between dark/light themes.
- Use the **search box** in the sidebar to filter commands.
- Click **Copy** on any code block to copy its contents.
